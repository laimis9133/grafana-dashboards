# grafana-ceph-pool-details

Grafana dashboard for Ceph pool details.

Pool available space and I/O actions are displayed. Physical and logical are separated.
![Here](https://github.com/laimis9133/grafana-dashboards/blob/main/ceph-pool-details/previews/example1.png?raw=true)

If pool quotas are set - they are displayed additionally and remaining space is calculated based on quota.
![Here](https://github.com/laimis9133/grafana-dashboards/blob/main/ceph-pool-details/previews/example2.png?raw=true)
