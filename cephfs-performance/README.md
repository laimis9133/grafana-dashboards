# grafana-cephfs-performance

Grafana dashboard for Ceph Filesystem Performance.

A ton of details displayed for each CephFS pool.
![Here](https://github.com/laimis9133/grafana-dashboards/blob/main/cephfs-performance/previews/example1.png?raw=true)

Focus for CephFS pool internals - MDS Journals, Memory etc. - that are pool-agnostic metrics.
![Here](https://github.com/laimis9133/grafana-dashboards/blob/main/cephfs-performance/previews/example2.png?raw=true)

Extra hidden Panel for OSD performance, cluster must have performance metrics enabled.
![Here](https://github.com/laimis9133/grafana-dashboards/blob/main/cephfs-performance/previews/example3.png?raw=true)
This is just an personal example of NVMe vs SSD performance, will have to finetune this section in the future for more generic cases.
